Verify Network Device Management State in Catalyst Center

Test Status: PASSED
Jobfile: tests/pyats/verify_dnac_network_device_management_state.py

Description

This test validates that all network devices discovered by Cisco Catalyst Center
are fully managed. The attributes managementState and collectionStatus reflect the
device's onboarding and inventory collection status.

Ensuring both are set to 'Managed' confirms that:
- Catalyst Center has successfully discovered, onboarded, and established management connectivity
- Inventory collection and monitoring are operational for each device

This validation is critical for network stability, allowing consistent configuration,
monitoring, and assurance across the managed device inventory.

Setup

  • Access to an active Cisco Catalyst Center (DNA Center) controller via HTTPS API
  • Valid authentication credentials for Catalyst Center
  • One or more network devices have been discovered and added to the Catalyst Center inventory
  • Device discovery and inventory collection have completed successfully

Procedure

  • Establish HTTPS connection to the Catalyst Center controller
  • Query the API endpoint: /dna/intent/api/v1/network-device
  • Retrieve the list of all discovered network devices
  • For EACH device in the response:
    • Extract the device's hostname and management IP address
    • Extract the managementState and collectionStatus attributes
    • Verify that both attributes are equal to 'Managed'

Pass/Fail Criteria

This test passes when all of the following conditions are met:

  • At least one network device is discovered in Catalyst Center inventory
  • ALL discovered devices have managementState equal to 'Managed'
  • ALL discovered devices have collectionStatus equal to 'Managed'

This test fails if any of the following criteria are met:

  • No network devices are discovered in the API response
  • ANY discovered device has managementState other than 'Managed'
  • ANY discovered device has collectionStatus other than 'Managed'
  • The API query fails or returns an error response
